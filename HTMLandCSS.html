HTML and CSS; how do they work together?

HTML

HTML (hyper text markup language) is used to define the structure of a webpage.

A HTML page is made up of elements which are defined using tags. Every website on the internet uses HTML to some extent.

When HTML is used on it's own, a website will look terrible Click Here to see this page using just HTML and no css

CSS

CSS (cascading style sheets) is used to describe how the HTML elements are represented on screen. Using CSS you can completely change how a webpage looks by changing the css code and completely leaving the HTML as it is.
